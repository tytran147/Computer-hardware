Introduction to Computer Hardware

The computer system consists of two types of components: software and hardware. The software components provide instructions for specific tasks, while the hardware components are the physical parts of the computer system.

Internal Components

Component Description
Computer Case A metal enclosure used to assemble and protect various internal hardware components.
Motherboard The main printed circuit board that interconnects various computer system components.
Microprocessor (CPU) The brain of the computer system, providing processing power.
Power Supply Unit (SMPS) Provides stable electrical power supply to various hardware components.
Random Access Memory (RAM) The main memory of the computer, used for storing data temporarily during program execution.
Graphics Card Handles graphics processing functions, improving the quality of graphics rendered by the computer system.
Hard Disk Provides long-term storage for data, managed by the operating system.
Solid State Drive (SSD) Offers faster data access speeds and higher durability compared to traditional hard drives.

Laptop Hardware Components

Laptops have gone through major technological changes, especially in motherboard design and the integration of hardware components. Despite these changes, the fundamental architecture remains similar to desktop computers.

  • Integrated Motherboard
  • Compact CPU and GPU
  • Battery
  • Integrated Display
  • Touchpad
  • Built-in Webcam
